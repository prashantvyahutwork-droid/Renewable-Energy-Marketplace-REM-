# BIJLI.GRID - Decentralized Energy Marketplace

BIJLI.GRID is a futuristic, decentralized peer-to-peer (P2P) energy trading platform. It allows users to trade energy directly with one another through a secured, blockchain-integrated interface.

## üìã Setup & Execution

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher)
- [npm](https://www.npmjs.com/) (usually comes with Node.js)
- [MetaMask](https://metamask.io/) (optional, for Web3 features)

### 1. Installation
Clone the repository and install dependencies:
```bash
git clone <repository-url>
cd Marketplace
npm install
```

### 2. Running Locally (Development)
Start the Vite development server:
```bash
npm run dev
```
Wait for the terminal to show the local URL (usually `http://localhost:5173`). Open this URL in your browser.

### 3. Usage Instructions
1. **Authentication**: Start at the **Identity** portal. You must **Sign Up** or **Log In** to unlock the application browsing panels.
2. **Dashboard**: Navigate to the Dashboard to see the project overview and core grid status.
3. **Marketplace**: Connect your MetaMask wallet, then browse active energy requests. Click **"Accept & Fulfill"** to initiate a transfer.
4. **Ledger**: Check the Ledger to see your successfully completed energy transactions.
